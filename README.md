# Budget & Expense Variance Tracker

## Project Overview
This Excel project tracks monthly budgeted vs. actual revenue and expenses for a small business. The goal is to identify budget variances, monitor spending, and support basic financial decision-making.

## Business Question
How closely are actual revenues and expenses aligning with the monthly budget?

## Tools Used
- Microsoft Excel
- Pivot Tables
- Formulas
- Charts
- Variance Analysis

## Key Features
- Budgeted vs. actual tracking
- Variance calculation
- Over/under budget status
- Expense category summary
- Dashboard with charts and KPIs

## Key Insights
- Identified categories that exceeded budget
- Compared planned and actual financial performance
- Summarized net profit impact from revenue and expense changes

- Financial reporting
- Budget tracking
- Variance analysis
- Excel formulas
- Dashboard creation
